- This is a simple system to register students for courses.
- Each time a student registers, the registration is added to the end of the waiting list.
- The administrator can:
    * Register a student
    * Remove a student
    * Search for a student
    * Display all registered students
    * Count registered students
  
PURPOSE OF THIS MINI PROJECT:
    - To gain hands-on-skill on Linked List and OOP Paradigm
